# 🚀 クイックスタートガイド

## 即座に始める3ステップ

### 1. プロジェクトに移動
```bash
cd english-learning-app
```

### 2. 依存関係をインストール
```bash
npm install
```

### 3. 開発サーバーを起動
```bash
npm run dev
```

ブラウザで http://localhost:3000 が自動的に開きます!

## 🎮 使い方の流れ

1. **最初にすること**: 「単語を追加」で学習したい単語を登録
2. **毎日の学習**: 「今日の学習」から学習開始
3. **復習**: エビングハウスの忘却曲線に基づいて自動的に復習日が設定される
4. **間違えた単語**: 学習後にコピーしてAIに貼り付け
5. **進捗確認**: 統計画面で自分の成長を確認

## 🔧 カスタマイズ

### テーマ色を変更したい

### バッジを追加したい
`src/constants.js` のBADGES配列に追加

## ⚠️ 注意事項

### データ保存について
- データはブラウザのPersistent Storageに保存されます
- ブラウザのキャッシュをクリアするとデータが消えます
- 定期的にエクスポート（バックアップ）を推奨します

### 対応ブラウザ
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### プライベートモード
- プライベートブラウジングモードでは動作しません
- データが保存されないためです

## 🐛 トラブルシューティング

### npm install でエラーが出る
```bash
# node_modules を削除して再インストール
rm -rf node_modules package-lock.json
npm install
```

### 開発サーバーが起動しない
```bash
# ポート3000が使用中の場合
npm run dev -- --port 3001
```

### データが保存されない
- ブラウザのストレージ設定を確認
- シークレットモードでないか確認
- 別のブラウザで試してみる
