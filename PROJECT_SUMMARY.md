## 🎯 データ構造

### 単語データ
```javascript
{
  id: "uuid",
  english: "take a rain check",
  japanese: ["また今度にする", "延期する"], // 複数訳対応
  primaryMeaning: "また今度にする",
  category: "idiom",
  example: "I'll take a rain check on dinner.",
  note: "カジュアルな断り方として使える",
  tags: ["ビジネス", "日常会話"],
  status: "話せる",
  // ... 学習データ
}
```

### フィールド制限
- 英語: 200文字
- 日本語訳: 500文字（全訳合計）
- 例文: 500文字
- メモ: **2000文字**
- タグ: 最大10個（各20文字）

## 💡 特別機能: 間違い単語リスト

学習後に間違えた単語をコピーして、Claude.aiやChatGPTに貼り付けるだけ!

### フォーマット例（AI用）
```
以下の英語表現を覚えるための効果的な学習方法を教えてください:

1. take a rain check
   意味: また今度にする

2. break the ice
   意味: 打ち解ける
```

→ これをClaude.aiに貼り付けると、学習方法のアドバイスが受けられます!

## ⚠️ 重要な注意事項

### データ保存について
- ブラウザのPersistent Storageに保存
- **ブラウザのキャッシュクリアでデータ消失**
- 定期的なエクスポート（バックアップ）を推奨

### 推奨の使い方
1. 週に1回はデータをエクスポート
2. JSON形式で保存（完全バックアップ）
3. CSV形式も便利（Excelで編集可）
