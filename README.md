# English Learning Cards 📚

エビングハウスの忘却曲線を活用した効率的な英語学習アプリ

## 🌟 主な機能

### コア機能
- **単語・慣用句管理**
  - 複数の日本語訳対応
  - 例文、メモ、タグ機能
  - 5段階ステータス（まだまだ→聞ける→話せる→書ける→マスター）

- **学習機能**
  - カード形式での学習
  - 3つの学習モード（英→日、日→英、ランダム）
  - 自己採点システム
  - エビングハウスの忘却曲線による復習スケジューリング

- **間違い単語リスト**
  - 学習後に間違えた単語を一覧表示
  - 3つのフォーマット（シンプル、AI用、CSV）
  - ワンクリックコピー機能
  - AI学習補助用のテンプレート

### 統計・モチベーション機能
- **基本統計**
  - 今日学習した件数
  - 今日の正答率
  - 総登録単語数
  - ステータス別分布

- **可視化**
  - 週間学習グラフ
  - 正答率推移グラフ
  - カレンダーヒートマップ（90日間）
  - カテゴリ別習得状況

- **モチベーション**
  - 学習ストリーク（連続学習日数）
  - レベル・経験値システム
  - バッジコレクション
  - やる気スコア
  - 苦手な単語TOP10

### カスタマイズ
- 復習間隔のカスタマイズ
- 1日の目標件数設定
- デフォルト学習モード設定

### データ管理
- JSON/CSV形式でのエクスポート
- ブラウザのPersistent Storageに保存
- データ削除機能

## 🚀 セットアップ手順

### 必要な環境
- Node.js 18以降
- npm または yarn

### インストール

1. **依存関係のインストール**
```bash
cd english-learning-app
npm install
```

2. **開発サーバーの起動**
```bash
npm run dev
```

3. **ブラウザで開く**
```
http://localhost:3000
```

### ビルド

本番環境用のビルド:
```bash
npm run build
```

ビルドしたアプリのプレビュー:
```bash
npm run preview
```

## 📱 使い方

### 1. 単語を登録
- ホーム画面から「単語を追加」をクリック
- 英語、日本語訳（複数可）、カテゴリなどを入力
- 必要に応じて例文、メモ、タグを追加

### 2. 学習する
- ホーム画面の「今日の学習」をクリック
- 学習モードを選択
- カードをタップして答えを確認
- ○×で自己採点

### 3. 間違えた単語を復習
- 学習完了後、「間違えた単語」をクリック
- お好みのフォーマットを選択
- 「コピー」ボタンでクリップボードにコピー
- Claude.aiやChatGPTに貼り付けて学習方法を聞く

### 4. 進捗を確認
- 統計画面で学習の軌跡を確認
- バッジコレクションでモチベーション維持
- カレンダーヒートマップで学習習慣を可視化

## 🎯 エビングハウスの忘却曲線

デフォルトの復習間隔:
```
正解時: 1日 → 2日 → 4日 → 7日 → 14日 → 30日 → 60日 → 90日...
不正解時: 翌日に再出題
```

設定画面でカスタマイズ可能!

## 🔧 技術スタック

- **フロントエンド**: React 18
- **スタイリング**: Tailwind CSS
- **グラフ**: Recharts
- **アイコン**: Lucide React
- **ビルドツール**: Vite
- **データ保存**: Persistent Storage API

## 📝 フィールド制限

- 英語: 200文字
- 日本語訳: 500文字（全訳合計）
- 例文: 500文字
- メモ: 2000文字
- タグ: 最大10個（各20文字）

## 🎨 カスタマイズ

### 復習間隔の変更
設定画面から自由にカスタマイズ可能

### テーマ色の変更
`tailwind.config.js`のカラー設定を編集

## 🐛 トラブルシューティング

### データが保存されない
- ブラウザのストレージ設定を確認
- プライベートブラウジングモードでは動作しません

### グラフが表示されない
- ブラウザを再読み込み
- 学習データが少ない場合は一部グラフが表示されません
